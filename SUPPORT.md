# Support

## Usage Help

For setup questions, operator workflow help, or troubleshooting:

- Open a GitHub Discussion (preferred) if enabled
- Or open a GitHub Issue with logs and reproduction steps

## Bug Reports

Please include:

- OS and app version
- Steps to reproduce
- Expected behavior vs. actual behavior
- Relevant logs from:
  `~/Library/Logs/com.church.live.translate/app.log` (macOS)

## Security Issues

For vulnerabilities, follow [SECURITY.md](./SECURITY.md) and use private reporting.
